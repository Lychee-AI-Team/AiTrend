# AiTrend

多源 AI 热点资讯收集器

## 功能

- 多数据源挖掘：Twitter、Product Hunt、HackerNews、GitHub、Brave Search、Reddit
- AI 智能总结：Gemini 3 Flash Preview
- 口语化内容生成

## 数据源

| 数据源 | 类型 |
|--------|------|
| Twitter/X | 实时讨论 |
| Product Hunt | 新产品发布 |
| HackerNews | 开发者分享 |
| GitHub | 开源项目 |
| Brave Search | 全网搜索 |
| Reddit | 社区讨论 |

## 运行

```bash
python3 -m src
```

## 配置

复制 `.env.example` 为 `.env`，填入 API Key

## 依赖

纯 Python 标准库，零外部依赖
