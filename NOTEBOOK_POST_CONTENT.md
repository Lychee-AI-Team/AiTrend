## 💬 互动回复模板

### 常见问题回复

**Q: 这个和 RSS 有什么区别？**
> A: RSS 只是聚合，AiTrend 会 AI 生成独特的中文介绍，自动去重，还能直接发布到 Discord 等渠道。更重要的是模块化设计，可以自由组合数据源和渠道。

**Q: 收费吗？**
> A: 完全免费开源！MIT 协议，可以商用、修改、分发。

**Q: 需要编程基础吗？**
> A: 不需要！改配置文件即可，有详细文档和一键安装脚本。

**Q: 支持哪些数据源？**
> A: 目前支持 GitHub Trending、Product Hunt、HackerNews、Reddit、Tavily，还在增加中。也欢迎贡献新的数据源！

**Q: 怎么部署？**
> A: 三种方式：1) 一键脚本安装 2) Docker 部署 3) OpenClaw Skill 安装
