## 报告安全漏洞

如果您发现了安全漏洞，请 **不要** 创建公开的 Issue。

### 报告方式

请发送邮件至：ai-lychee@example.com

邮件内容请包含：
1. 漏洞描述
2. 复现步骤
3. 可能的影响
4. 建议的修复方案（如果有）

### 响应时间

- **确认收到**: 24 小时内
- **初步评估**: 72 小时内
- **修复发布**: 取决于严重程度

## 安全最佳实践

### API Key 保护

- 永远不要将 API Key 提交到 Git
- 使用 `.env.keys` 文件存储密钥
- 定期轮换 API Key
- 使用最小权限原则

### 代码安全

- 所有用户输入都经过验证
- 避免命令注入风险
- 使用安全的 HTTP 客户端
